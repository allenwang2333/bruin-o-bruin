inspired by 羊了个羊  

## Run locally
```
git clone git@github.com:allenwang2333/bruin-o-bruin.git
cd bruin-o-bruin
npm install
npm run dev
cd bruin-o-bruin
npm install
npm run build
```  

Then, check http://localhost:8080/ to see the web-app
